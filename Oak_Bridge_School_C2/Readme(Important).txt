This project is done while doing course in NIIT
This project focuses on managing and analyzing students' marks at OakBridge School. It covers:

Calculating total and average marks for each student based on their scores in math, science, and English.
Assigning grades to students based on their marks in each subject.
Sorting students by their total marks using insertion sort.
Checking for repeated student names.
Finding the top score using recursion.
Generating and displaying student reports, including their total marks, average marks, and grades.
Key concepts:

Arrays: Store and manipulate student marks, grades, and names.
Sorting Algorithm: Insertion sort is used to arrange students by their total marks.
Recursion: Used to find the top score among students.
String Manipulation: Builds result messages for each student.
